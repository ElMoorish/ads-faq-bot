# About TikTok Ads Manager Video Editor
## Official Documentation

---

The TikTok Ads Manager Video Editor is an advanced editing solution tailored for advertisers found within TikTok Ads Manager. This editor helps advertisers efficiently enhance ad performance right within TikTok Ads Manager's ad creation process.

---

## Key Features

### 1. AI-Assisted Narration
- On-screen narration of your video script
- Add a visual avatar
- Customizable options
- Enriches storytelling
- Improves viewer engagement

**Learn more:**
- Generate voiceovers: https://ads.tiktok.com/help/article/how-to-generate-voiceovers-with-tiktok-video-editor
- Create avatars: https://ads.tiktok.com/help/article/how-to-create-avatars-with-tiktok-video-editor

---

### 2. Multilingual Video Adaptation
- Translation features
- Caption features
- Localize videos into multiple languages
- Broadens reach
- Enhances accessibility for international viewers

**Learn more:** https://ads.tiktok.com/help/article/how-to-generate-captions-with-tiktok-video-editor

---
